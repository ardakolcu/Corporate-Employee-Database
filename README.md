# Corporate Employee Salary Tracking Database

## About The Project
This repository contains the relational database architecture and SQL scripts for a comprehensive Employee Salary Tracking Program. It was designed to manage corporate human resources data, including employee details, departmental hierarchies, and salary processing.

## Database Architecture
The project includes full documentation of the database design process:
* **Conceptual Model:** High-level entity relationships.
* **Logical Model:** Detailed attributes, data types, and primary/foreign keys.
* **Physical Model:** Executable SQL Server architecture.

Key entities structured in this project include: `Employees`, `Salaries`, `Departments`, `Users`, and `Categories`.

## Technologies Used
* **Microsoft SQL Server 2022**
* **T-SQL** (DDL & DML scripts for database creation and data insertion)
* Relational Database Design & Architecture

## Files in this Repository
* `Corporate_Employee_Database.sql`: The main SQL script to generate the database, tables, relational constraints, and sample records.
* `Database_Architecture_Report.pdf`: The technical documentation containing the ER diagrams and data dictionaries.
